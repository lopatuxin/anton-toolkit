# Niche context — IT / vibe-coding channel

Use this as the worldview when generating ideas. It is the channel's positioning, not free-form imagination.

## Channel identity

- Russian-language IT channel.
- Theme: **vibe coding** — AI-assisted application development.
- Tools the channel actively uses: Cursor, Claude Code, Codex, similar AI-coding tooling. Update this list when the user mentions a new tool.
- Format: tutorials, build-alongs, case studies, occasional opinion.
- Cadence: 1 video / week, ≤ 20 minutes. No Shorts.

## Flagship series

- **Logos / Логос** — building a Jarvis-like personal AI assistant. Long-running series. Each episode adds a capability or tackles an obstacle. Continuity matters.
- **CRM for auto-services** — full-stack AI-assisted business app, end-to-end series.

When generating series-continuation ideas, the next episode should:

- Build on the previous episode's state (do not reset).
- Be self-contained enough to watch standalone (catches new viewers).
- Either add a capability, fix a hard problem, or refactor with a visible upgrade.

## Audience

- Russian-speaking developers, mid-junior to senior.
- Interested in AI-assisted development as a productivity multiplier and as a business opportunity.
- Two intents:
  - Learn to use AI-coding tools effectively (tutorial / how-to demand).
  - Find a contractor for their own AI-assisted project (lead-gen target).

## Lead-gen target

The channel's commercial goal: **orders for application development**.

When tagging an idea as lead-gen-anchor:

- It must show the kind of work the user can deliver.
- It must end with a CTA that points at the business-card site (currently `{{SITE_URL}}` placeholder; soft fallback: "контакты в закреплённом комментарии").
- The CTA must feel earned by the content, not bolted on.

## Tone

- Practical, opinionated, no hype.
- "Вот что я сделал и почему" beats "вот как делать".
- Russian-speaking IT audience reacts negatively to overpromise. Honest framing wins.

## What NOT to propose

- Generic "AI for beginners" content — the channel is past that.
- Tool reviews without a strong personal angle — the niche is saturated.
- "10 things you didn't know about <tool>" listicles — low retention in this niche.
- Anything requiring on-camera polish above what the user produces today (filming style is not in scope).
