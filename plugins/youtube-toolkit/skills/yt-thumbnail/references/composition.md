# Composition rules — text, layers, colour, emotion

## Text overlay

- **3–5 words.** Three is often best. Above five, the eye skips.
- **Russian language.** Tool names in original form (Cursor, AI).
- **No full sentences.** Fragments hit harder: "Cursor сломался." beats "Почему Cursor сломался у меня".
- **Punctuation rules:**
  - Period — adds finality. Use for opinions and verdicts.
  - Question mark — adds curiosity. Use for problem-mirror.
  - No exclamation marks. Cheapens.
- **Capitalisation:** sentence case or ALL CAPS. Consistent across the channel. ALL CAPS reads bolder; sentence case reads more confident. Pick one for the channel and stick.
- **Number rule:** if a number is in the text, use digits (`2 дня`, `47 раз`), not words.

## Layers

Three-layer model:

1. **Background:** sets context. Often blurred or darkened so foreground stands out. Examples: blurred IDE, dim gradient, abstract gradient with no detail.
2. **Mid:** the "thing" — the screenshot, the artefact, the comparison. This carries 50–70% of the visual weight in patterns B, C, E.
3. **Foreground:** face, large text, or a focal element. Carries the emotion or the headline.

In pattern A (face+screen split), foreground is the face *and* the headline text together; mid is the screenshot.

## Colour scheme

- **2–3 colours max.** Plus black/white for text and edges.
- **One dominant** (60–70% of canvas), one accent (20–30%), one contrast for the text.
- Dark backgrounds (deep blue, near-black, dark purple) work for IT channels — they imply "professional, screen-based work".
- Avoid: high-saturation reds and yellows on most of the canvas. They signal sensationalism that this niche reads as untrustworthy.
- The series stamp (pattern E) freezes its colour scheme on episode 1. Never recolour mid-series.

## Emotion

Pick **one** dominant emotion. Listed by hit-rate in this niche:

1. **Curiosity** — slight smile, eyebrows up, looking at something off-frame. Default for tutorials and tool reviews.
2. **Triumph / proof** — controlled, not exaggerated. "I made this" energy. For case studies and lead-gen.
3. **Frustration / disbelief** — exasperated, hand on head. For opinion and "why I stopped using X" videos. Use sparingly.
4. **Focus** — neutral, looking at screen, intense. For deep-dive series episodes.

Avoid: the open-mouth shock face. Saturated to the point of being a meme.

## Legibility tests

Before approving a concept, mentally check:

- **Mobile test:** at thumbnail size on a phone (240×135 px), can the text be read in 1 second?
- **Sidebar test:** in YouTube's "Up next" sidebar, the thumbnail is even smaller. Top-left zone is most visible — keep the highest-contrast element there.
- **Coverage test:** if you covered the bottom 60% of the canvas, does the top still convey the hook?

If any of these fail, revise.

## Output discipline

- Produce 2–3 variants per video. Each variant must use a *different* pattern or a *different* text angle.
- Variants are not for "more options" — they're for letting the user pick the one that matches the recording they actually got.
- For each variant, name the pattern explicitly. Patterns are reusable; novel one-offs aren't.
