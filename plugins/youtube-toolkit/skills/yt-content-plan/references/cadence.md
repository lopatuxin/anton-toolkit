# Cadence and balance rules

## Hard constraints

- **1 video per week.** No exceptions. The channel cannot sustain more.
- **≤ 20 minutes per video.**
- **No Shorts.** If an idea is a Short, drop it.

## Mix targets (per 8-week window)

| Category | Target slots |
|---|---|
| Flagship-series episode (Logos / CRM / other named series) | 3–4 |
| Lead-gen case study | 2 |
| Tutorial / how-to | 1–2 |
| Tool review / opinion | 0–1 |

Adjust if the idea pool skews — the rule is "balance toward this target", not "force exactly".

## Adjacency rules

- **Never two flagship series episodes back-to-back of the same series.** Logos → Logos → tutorial → CRM is fine; Logos → Logos is not (locks in subscribers, loses casual viewers).
- **Never two lead-gen videos back-to-back.** The CTA fatigue is real.
- **Open the period with a top-funnel friendly idea.** Tutorial or tool-angle case study, not a deep series episode that requires prior context.

## Spacing rules

- Spread flagship-series episodes evenly. If you place 4 in 8 weeks, target weeks 1, 3, 5, 7.
- Place lead-gen 3–4 weeks apart.
- Tool review or opinion videos are filler — use them to break adjacency violations.

## Production status

For each scheduled video, mark one of:

- `idea` — only the idea exists
- `script` — script being written or done
- `recording` — recorded, in edit
- `editing` — in edit
- `ready` — ready to publish

A healthy plan has ≥ 2 weeks of "ready" or "editing" buffer. Flag if buffer < 2.

## Dependencies

Some ideas depend on a previous one (e.g., Logos episode N+1 needs episode N out). Mark dependencies explicitly so the calendar enforces the order.

## When the idea pool is short

If the period has more weeks than ideas:

- Do not duplicate ideas to fill.
- Mark the empty weeks `<gap — need new idea>`.
- Note in the output: "Нужно ещё N идей — запусти yt-ideas с обновлённым отчётом по конкурентам."
