# Strategy Templates — шаблоны стратегий и дорожных карт

## Формат файла стратегии (strategy/active-strategy.md)

YAML-фронтматтер:
```
---
last_updated: YYYY-MM-DD
confidence: 0.X
status: active | paused | completed | abandoned
target_date: YYYY-MM-DD
---
```

Структура:
```
# Стратегия: [Название]

## Цель
Конкретная формулировка цели по SMART:
- **Specific:** Что именно
- **Measurable:** Как измерить
- **Achievable:** Почему реалистично (с учётом профиля)
- **Relevant:** Почему важно (связь с ценностями из профиля)
- **Time-bound:** К какому сроку

## Текущая позиция
Где пользователь сейчас относительно цели. Ссылки на конкретные данные из профиля.

## Ключевые рычаги
Что даст максимальный эффект с учётом сильных сторон и ресурсов пользователя.

## Риски и препятствия
С учётом известных паттернов (из decisions/patterns.md) и ограничений.

## План действий
Конкретные шаги, привязанные к срокам.

## Метрики прогресса
Как отслеживать — конкретные индикаторы.
```

## Формат дорожной карты (strategy/roadmap.md)

```
# Дорожная карта

## Q[N] YYYY — [Тема квартала]

### Веха 1: [Название] — до DD.MM
- [ ] Шаг 1: [конкретное действие]
- [ ] Шаг 2: [конкретное действие]
- **Метрика:** [как проверить]
- **Зависимости:** [что нужно до этого]

### Веха 2: [Название] — до DD.MM
- [ ] ...
```

## Принципы построения стратегий

1. **Персонализация** — каждая рекомендация опирается на конкретные данные из профиля
2. **Учёт когнитивного стиля** — для аналитиков: метрики и данные; для интуитивов: общая картина и ощущения
3. **Учёт мотивации** — для мотивации достижения: фокус на целях; для мотивации избегания: фокус на рисках бездействия
4. **Маленькие шаги** — декомпозиция до уровня «это можно сделать за 1 день/неделю»
5. **Буферы** — закладывать 20-30% запаса по времени
6. **Точки пересмотра** — встроенные моменты для оценки «а правильно ли я иду?»
7. **Антихрупкость** — стратегия должна работать даже при частичном провале

## Антипаттерны

- Общие советы без привязки к профилю («просто начни делать»)
- Нереалистичные сроки без учёта ограничений
- Игнорирование паттернов прокрастинации/избегания из профиля
- Предложение подходов, конфликтующих с личностными особенностями
- Перегрузка параллельными целями без приоритизации
