# Profile Review — ревью и обновление профиля

Проведи полный аудит профиля, найди пробелы и устаревшие данные, уточни информацию.

## Процесс

### 1. Проверь наличие профиля

```bash
ls -d /c/projects/personal-profile ~/projects/personal-profile 2>/dev/null
```

Если профиль НЕ существует — предложи сначала пройти онбординг.

### 2. Запусти аналитика

Запусти агента `ps-analyst`:

> Проведи полный аудит профиля. Определи:
> - Области с низким confidence (< 0.5)
> - Данные старше 30 дней
> - Пустые или минимально заполненные секции
> - Противоречия и несоответствия

### 3. Уточни пробелы

На основе аудита запусти `ps-interviewer` для областей с пробелами:

> Проведи уточняющее интервью. Фокус на следующих областях:
> [список областей с низким confidence или устаревшими данными из анализа]
> Не нужно повторять уже известную информацию — уточняй только пробелы.

### 4. Обнови profile-summary

После уточнения запусти `ps-analyst` снова:

> Обнови meta/profile-summary.md на основе новых данных.
> Обнови meta/last-review.md с текущей датой и результатами.

### 5. Покажи результат

Покажи пользователю:
- Что обновилось
- Какие области теперь хорошо покрыты
- Что ещё осталось с низким confidence
- Когда рекомендуется следующий ревью
