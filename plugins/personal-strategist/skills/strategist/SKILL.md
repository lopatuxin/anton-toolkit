# Strategist — стратегический советник

Разработай стратегию или дай рекомендации на основе профиля пользователя.

## Процесс

### 1. Проверь наличие профиля

```bash
ls -d /c/projects/personal-profile ~/projects/personal-profile 2>/dev/null
```

Если профиль НЕ существует — предложи сначала пройти онбординг.
Если профиль существует — продолжай.

### 2. Запусти аналитика

ВСЕГДА сначала запусти агента `ps-analyst`:

> Проведи анализ текущего профиля. Пользователь хочет: [суть запроса].
> Обрати особое внимание на: [релевантные категории].

Дождись результата анализа.

### 3. Запусти стратега

На основе анализа запусти `ps-strategist`:

> На основе анализа разработай стратегию для: [суть запроса].
> Результат анализа: [вставь результат ps-analyst].
> Учти запрос пользователя: [оригинальный запрос].

### 4. Покажи результат

Стратег обсудит стратегию с пользователем и запишет одобренный вариант в профиль.

## Быстрый режим

Если запрос простой (один конкретный совет, а не полная стратегия):
- Прочитай `meta/profile-summary.md` и релевантные файлы сам
- Дай рекомендацию с привязкой к профилю
- Не запускай полный цикл analyst → strategist

Определяй режим по сложности запроса:
- «Как мне построить карьеру в AI?» → полный цикл
- «Стоит ли мне пойти на эту конференцию?» → быстрый режим
